# Code of Conduct

This project follows the [Contributor Covenant v2.1][covenant].

In short: be respectful, be constructive, assume good intent. Technical
disagreements are fine; personal attacks are not.

To report a violation, email **conduct@legalize.dev**. Reports are
handled confidentially.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
